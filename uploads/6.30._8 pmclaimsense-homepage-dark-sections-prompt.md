# ClaimSense Homepage — Tmavé pozadí + přesun stats sekce

## Kontext: kde se nacházíme ve stránce

Aktuální pořadí sekcí (shora dolů):

```
1. Hero
2. Use Cases across industries     ← pozadí #FFE4DF (teplá růžová)
3. [VIDEO PLACEHOLDER sekce]       ← aktuálně světlé / bílé pozadí
4. Built by Bull + stats (5K+, 28, 14%)  ← pozadí #FFE4DF
5. Testimonials
6. Before / After

9. Pricing

```

---

## ZMĚNA 1 — Tmavé pozadí pro video placeholder sekci

Najdi sekci obsahující video placeholder mezi sekcí "Use Cases across industries" a sekcí "Built by Bull". Identifikátorem je přítomnost:
- `client.claimsense.app` URL v browser chrome mockupu, NEBO
- třídy nebo ID odkazující na demo video / product screenshot

Přidej tomuto elementu (nebo jeho nejbližšímu rodičovskému `<section>` / wrapper divu) třídu `section--dark` a aplikuj:

```css
.section--dark {
  background-color: #002870;
  padding-top: 48px; /* navazuje těsně na stats sekci bez mezery */
}

/* Veškerý text uvnitř tmavé sekce musí být čitelný: */
.section--dark h1,
.section--dark h2,
.section--dark h3,
.section--dark p {
  color: #ffffff;
}

.section--dark p {
  color: rgba(255, 255, 255, 0.75);
}
```

Nic jiného v sekci neměň — jen pozadí a barvy textu.

## ZMĚNA 2 — Přesun a redesign stats sekce (10x / 70% / 100+ / 96%)

### Krok 1: Přesuň sekci

Najdi sekci "What better claims operations look like" obsahující čtyři metrické karty (10x, 70%, 100+, 96%). Tuto celou sekci **přesuň** v HTML tak, aby se nacházela **přímo za** sekcí "Use Cases across industries" (sekce s růžovým pozadím #FFE4DF).

Výsledné pořadí po přesunu:
```
2. Use Cases across industries   ← #FFE4DF, beze změny
3. Stats sekce (10x...)          ← přesunuto sem, tmavá modrá ← NOVÁ POZICE
4. [VIDEO PLACEHOLDER sekce]     ← tmavá modrá (Změna 1)
5. Built by Bull + stats...
...
```

### Krok 2: Redesign layoutu na west-east (jedna řada)

Aktuální layout stats sekce: karty jsou v gridu se světlými sub-kartami uvnitř tmavé sekce.
