# ClaimSense — Master CTA Button System Prompt

> Tento prompt definuje chování a styling VŠECH interaktivních tlačítek a štítků napříč celým projektem.
> Neměň tvary tlačítek. Měň pouze barvy, barvu textu a hover stavy přesně podle níže uvedených pravidel.

---

## PRAVIDLO 0 — Obecné pravidlo pro text uvnitř oranžových tlačítek

Kdekoli má tlačítko oranžový podklad (`background-color: #FF5539`):
- Barva textu musí být **černá** (`#000000`), ne bílá
- Platí pro DEFAULT i HOVER stav
- Neměň font, velikost ani tvar

```css
/* Globální pravidlo — oranžový button = černý text */
[class*="btn"]:not([class*="outline"]):not([class*="ghost"]):not([class*="blue"]),
.btn-primary,
.button-orange,
[style*="background-color: rgb(255, 85, 57)"],
[style*="background-color: #FF5539"] {
  color: #000000 !important;
}
```

## PRAVIDLO 1 — Primární CTA tlačítko (oranžový podklad)

Použití: "Book a walkthrough", "Book a Demo", "Get started" na oranžovém podkladu — hlavní konverzní CTA kdekoliv na stránce.

```
DEFAULT:  oranžový podklad (#FF5539) + černý text (#000000)
HOVER:    bílý podklad + oranžový border (2px solid #FF5539) + černý text (#000000)
```

```css
.btn-primary {
  background-color: #FF5539;
  color: #000000;
  border: 3px solid #FF5539;
  transition: background-color 0.2s ease, color 0.2s ease;
}

.btn-primary:hover {
  background-color: #ffffff;
  color: #000000;
  border: 3px solid #FF5539;
}
```

Použití: "Get started" v Pricing sekci — varianta Quick Start (bílá karta).

.btn-secondary {

  transition: border-color 0.2s ease;

## PRAVIDLO 3 — Filtrační štítek — vybraný (modrý filled)

Použití: aktuálně vybraný filtr v "Across Industries" sekci na homepage (např. "Insurance"), aktuálně vybraný filtr na Use Cases stránce.

Štítek je **filtrační** — přepíná zobrazený obsah na místě, nepřesouvá uživatele jinam.

```
DEFAULT:  modrý podklad (#002870) + bílý text (#ffffff)
HOVER:    modrý podklad (#002870) + oranžový border (2px solid #FF5539) + bílý text
```

```css
.filter-tag--selected {
  background-color: #002870;
  color: #ffffff;
  border: 3px solid #002870;
  transition: border-color 0.2s ease;
}

.filter-tag--selected:hover {
  border-color: #FF5539;
  color: #ffffff;
  background-color: #002870;
}
```

## PRAVIDLO 4 — Filtrační štítek — nevybraný (outline / ghost)

Použití: nevybrané filtrovací štítky v "Across Industries" na homepage (All, Logistics & transport, Healthcare & pharma…), nevybrané filtrovací štítky na Use Cases stránce.

Štítek je **filtrační** — po kliknutí se stane vybraným (přejde do stavu Pravidla 3 nebo 5 podle stránky).

```
DEFAULT:  průhledný podklad + tmavý border (#002870) + tmavý text (#002870)
HOVER:    průhledný podklad + oranžový border (#FF5539) + oranžový text (#FF5539)
```

```css
.filter-tag {
  background-color: transparent;
  color: #002870;
  border: 1.5px solid #002870;
  transition: border-color 0.2s ease, color 0.2s ease;
}

.filter-tag:hover {
  border-color: #FF5539;
  color: #FF5539;
  background-color: transparent;
}
```

## PRAVIDLO 5 — Filtrační štítek — vybraný na Use Cases stránce (oranžový filled)

Použití: vybraný filtr na Use Cases stránce. Liší se od homepage tím, že selected stav je oranžový (ne modrý) — vizuálně výraznější signál aktivního filtru.

```
DEFAULT (nevybraný):  viz Pravidlo 4
HOVER (nevybraný):    průhledný + oranžový border + oranžový text
VYBRANÝ:              oranžový podklad (#FF5539) + černý text (#000000)
HOVER na vybraném:    oranžový podklad + oranžový border — beze změny
```

```css
/* Nevybraný — stejné jako Pravidlo 4 */
.use-case-tag {
  background-color: transparent;
  color: #002870;
  border: 1.5px solid #002870;
  transition: all 0.2s ease;
  cursor: pointer;
}

.use-case-tag:hover {
  border-color: #FF5539;
  color: #FF5539;
  background-color: transparent;
}

/* Vybraný */
.use-case-tag.is-active,
.use-case-tag[aria-selected="true"] {
  background-color: #FF5539;
  color: #000000;
  border-color: #FF5539;
}

.use-case-tag.is-active:hover {
  background-color: #FF5539;
  color: #000000;
  border-color: #FF5539;
}
```

## PRAVIDLO 6 — Filtrační pills na Features stránce (hover only, žádný persistentní vybraný stav)

Použití: pills 01–05 v horní části Features stránky (AI Pre-processing, Agentic workflow designer…).

Tyto pills jsou **filtrační** — kotví na sekci stránky. Na rozdíl od Use Cases štítků zde **není persistentní vybraný stav** — oranžová se zobrazí pouze při hoveru, po odjetí myší se vrátí do výchozího stavu.

.feature-pill {

## CO NEMĚNIT

- Tvary tlačítek (border-radius) — zachovej přesně jak jsou
- Velikosti písma a padding — beze změny
- Font-family — beze změny
- Transitions delší než 0.2s — nezkracuj, neprodlužuj
- Žádné shadow efekty nepřidávej ani neodebírej

## KDE APLIKOVAT

Projdi všechny HTML soubory projektu a aplikuj tato pravidla na:
- `index.html` (homepage) — primární CTA, filtrační štítky Across Industries
- `features.html` — feature pills 01–05
- `use-cases.html` — filtrační štítky
- `pricing.html` — Get started tlačítka
- Sdílený CSS soubor (global.css / styles.css / main.css) — přidej pravidla globálně a odstraň případné konflikty v page-specific souborech

## PRAVIDLO 7 — Zákaz stínování + "See full pricing & packages"

### Box-shadow: none globálně

Developer přidal stínování které není součástí Bull brand guidelines. Odstraň ho globálně:

```css
button,
[class*="btn"],
[class*="pill"],
[class*="tag"],
a[class*="btn"] {
  box-shadow: none !important;
}

button:hover,
[class*="btn"]:hover,
[class*="pill"]:hover,
[class*="tag"]:hover {
  box-shadow: none !important;
}
```

### "See full pricing & packages" tlačítko

Tmavě modrý filled button pod pricing kartami. Hover = oranžový border, podklad zůstane modrý:

```css
.btn-pricing-cta {
  background-color: #002870;
  color: #ffffff;
  border: 3px solid #002870;
  box-shadow: none;
  transition: border-color 0.2s ease;
}

.btn-pricing-cta:hover {
  border-color: #FF5539;
  background-color: #002870;
  color: #ffffff;
  box-shadow: none;
}
```
