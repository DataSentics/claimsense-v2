# ClaimSense Homepage — Tmavé pozadí video sekce + hero stats

## Co se mění a co se nemění

**NEMĚNÍ SE:**
- Sekce "What better claims operations look like" (10x / 70% / 100+ / 96%) zůstává přesně tam kde je, beze změny

**MĚNÍ SE:**
- Video placeholder sekce dostane tmavé pozadí `#002870`
- Nad video placeholder se přidají tři čísla z hero sekce jako vizuální hlavička

---

## ZMĚNA 1 — Tmavé pozadí pro video placeholder sekci

Najdi sekci obsahující video placeholder / product screenshot. Identifikátorem je:
- `client.claimsense.app` URL v browser chrome mockupu, NEBO
- třída nebo ID odkazující na demo video / product screenshot

Přidej nejbližšímu rodičovskému `<section>` třídu `section--dark` a aplikuj:

```css
.section--dark {
  background-color: #002870;
  padding: 64px 40px 72px;
}

.section--dark h1,
.section--dark h2,
.section--dark h3 {
  color: #ffffff;
}

.section--dark p {
  color: rgba(255, 255, 255, 0.75);
}
```

---
